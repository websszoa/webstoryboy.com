# 개발 환경

Runzoa 튜토리얼을 로컬에서 실행하는 방법입니다.

## 로컬 실행

```bash
npm run dev
```

`http://localhost:3000`에서 확인할 수 있습니다.

## Supabase 설정

1. [Supabase](https://supabase.com)에서 프로젝트 생성
2. 대회 데이터용 테이블 생성 (SQL Editor)
3. Authentication 설정

## 폴더 구조

- `app/` — 페이지 및 라우트
- `components/` — UI 컴포넌트
- `lib/` — Supabase, 스크래핑 유틸

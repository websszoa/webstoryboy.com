# 설치

Runzoa 튜토리얼을 진행하기 전 개발 환경을 준비합니다.

## 필요 조건

- Node.js 18.x 이상
- npm 또는 pnpm

## 프로젝트 생성

```bash
npx create-next-app@latest my-runzoa --typescript --tailwind --app
```

## 의존성 설치

```bash
cd my-runzoa
npm install @supabase/supabase-js @supabase/ssr
```

## 환경 변수

`.env.local`에 Supabase URL과 anon key를 넣어 주세요.

# Runzoa 소개

Runzoa는 전국 마라톤·러닝 대회 정보를 수집하고 검색할 수 있는 서비스입니다.

## 주요 기능

- **대회 검색**: 지역, 날짜, 거리로 필터링
- **대회 상세**: 코스, 참가비, 신청 링크 제공
- **즐겨찾기 & 댓글**: 로그인 시 이용 가능

## 기술 스택

Next.js 16, Supabase, Tailwind CSS, 웹 스크래핑을 활용했습니다.

## 튜토리얼 구성

1. **프로젝트 셋업** — Next.js와 Supabase 연동
2. **데이터 수집** — 대회 데이터 스크래핑 및 DB 저장
3. **UI 구현** — 목록, 검색, 상세 페이지
4. **인증 & 부가 기능** — 즐겨찾기, 댓글

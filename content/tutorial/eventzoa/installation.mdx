# 설치

튜토리얼을 따라 하기 전에 개발 환경을 준비합니다.

## 필요 조건

- Node.js 18.x 이상
- npm 또는 pnpm

## 프로젝트 생성

```bash
npx create-next-app@latest my-eventzoa --typescript --tailwind --app
```

## 의존성 설치

```bash
cd my-eventzoa
npm install @supabase/supabase-js @supabase/ssr
```

## 환경 변수

`.env.local`에 Supabase URL과 anon key를 설정합니다.

# 개발 환경

로컬에서 Eventzoa 튜토리얼을 실행하는 방법입니다.

## 로컬 실행

```bash
npm run dev
```

`http://localhost:3000`에서 확인할 수 있습니다.

## Supabase 설정

1. [Supabase](https://supabase.com)에서 프로젝트 생성
2. SQL Editor로 테이블 생성
3. Authentication에서 이메일/소셜 로그인 설정

## 폴더 구조

- `app/` — 페이지 및 라우트
- `components/` — 재사용 컴포넌트
- `lib/` — Supabase 클라이언트, 유틸
